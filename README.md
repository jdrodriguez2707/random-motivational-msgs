## Install

```npm
npm install -g random-motivational-msgs
```

## Usage

```bash
motivate
```

## Bonus

You can add this command to your `.bashrc` or `.zshrc` file to get a random motivational message every time you open a new terminal window.

```bash
echo "motivate" >> ~/.bashrc
```

```zsh
echo "motivate" >> ~/.zshrc
```
